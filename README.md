# База вопросов и ответов по Renga

Первая версия сайта FAQ в стиле макета с тёмной боковой навигацией и правой панелью подробного ответа.

## Состав

- `index.html` — главная страница
- `style.css` — оформление
- `script.js` — логика поиска, фильтров и карточек
- `data.js` — данные вопросов
- `data.json` — резервная копия данных
- `assets/` — папки для будущих PDF, GIF, изображений и файлов-примеров

## Размещение на GitHub Pages

Загрузите содержимое папки в репозиторий и включите GitHub Pages для ветки `main`, папка `/root`.


## GIF и изображения в ответах

Чтобы показать GIF или изображение прямо внутри открытой карточки вопроса:

1. Положите файл в папку `assets/gif` или `assets/img`.
2. В `data.js` или в Excel в поле `Файл-пример` укажите относительный путь, например:
   `assets/gif/GIF_ing.gif`
3. После обновления `data.js` сайт автоматически покажет GIF/изображение внутри карточки.

В этой версии пример GIF добавлен к вопросу `REN-Q-001`.

## Админ-панель

В архив добавлена страница `admin.html`.

Она позволяет:
- добавить новый вопрос через форму;
- автоматически определить следующий ID;
- выбрать существующий раздел или создать новый;
- сформировать готовый блок для вставки в `data.js`;
- скачать обновлённый `data.js` и `data.json`.

Важно: GitHub Pages и локальный HTML-сайт не могут сами сохранять изменения в файл `data.js`. После генерации записи нужно заменить файл в проекте вручную или загрузить скачанный `data.js` в репозиторий.
